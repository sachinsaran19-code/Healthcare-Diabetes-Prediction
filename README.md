# Healthcare Diabetes Prediction

## Project Overview
This project predicts diabetes risk using machine learning techniques on the Pima Indians Diabetes Dataset.

## Dataset
Pima Indians Diabetes Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models
1. Logistic Regression
2. Random Forest

## Results
- Logistic Regression Accuracy: 77.27%
- Random Forest Accuracy: 88.31%

## Key Findings
- Insulin was the most important feature.
- Glucose strongly influenced diabetes prediction.
- Random Forest performed better than Logistic Regression.

## Ethical Considerations
- Patient privacy must be protected.
- Predictions should support doctors, not replace them.
